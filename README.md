# Spotify Web API Testing Project

## Overview

This project demonstrates REST API testing using Spotify Web API through Postman.

The objective was to validate API functionality, authentication flow, response structure, status codes, and business logic using manual and exploratory testing techniques.

---

## Scope of Testing

- Tested REST APIs using:
  - GET
  - POST
  - PUT
  - DELETE

- Validated:
  - Basic Authentication
  - Access Token Generation
  - JSON Response Body
  - HTTP Status Codes
  - Headers & Parameters
  - Error Handling
